# Dentora Dental Clinic

Landing page for Dentora — a family-friendly dental care website.

**Private project.** All rights reserved. This is not open source. Do not copy, distribute, or use without permission.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```
